# Core Schemas

This file defines the structured JSON outputs used by different agents.

All LLM outputs that affect program control flow should follow these schemas as much as possible.

---

## MetaControllerOutput

Used by `MetaController` to decide which mode and agent should handle the current user request.

```json
{
  "mode": "fast_answer | clarify | react_tool | deep_review | offline_evolution",
  "next_agent": "chat_agent | clarification_agent | search_agent | file_agent | problem_agent | plan_agent | review_agent",
  "need_memory": true,
  "need_file": false,
  "need_search": false,
  "tools": [],
  "reason": "",
  "risk_level": "low | medium | high"
}
```

---

## ClarificationOutput

Used by `ClarificationAgent` to decide whether to ask the user follow-up questions.

```json
{
  "need_clarification": true,
  "clarification_question": "",
  "reason": "",
  "assumption_if_no_question": ""
}
```

---

## ProblemOutput

Used by `ProblemAgent` or `ProblemDiscoveryAgent` to describe discovered problems.

```json
{
  "problems": [
    {
      "problem_type": "knowledge_gap | method_gap | planning_issue | execution_issue | emotion_issue | cognitive_bias | project_design | other",
      "subject": "",
      "description": "",
      "evidence": [],
      "root_cause": "",
      "severity": 1,
      "confidence": 0.0,
      "value_score": 1,
      "suggested_action": "",
      "status": "open"
    }
  ],
  "overall_summary": ""
}
```

---

## MemoryOutput

Used by `MemoryAgent` to decide whether a candidate memory should be stored, updated, merged, or skipped.

```json
{
  "should_store": true,
  "operation": "insert | update | merge | skip",
  "memory_type": "preference | learning_status | weakness | mistake_pattern | intervention_result | project_state | strategy",
  "content": "",
  "importance": 1,
  "confidence": 0.0,
  "merge_key": "",
  "reason": ""
}
```

---

## FileRecord

Used by `FileAgent` to store metadata for uploaded files.

```json
{
  "file_id": "",
  "filename": "",
  "file_type": "",
  "storage_path": "",
  "text_path": "",
  "summary": "",
  "subject": "",
  "tags": [],
  "created_at": ""
}
```

---

## DailyReview

Used by `ReviewAgent` to generate a daily study review.

```json
{
  "review_date": "",
  "summary": "",
  "learned_today": [],
  "problems_found": [],
  "recommended_actions": [],
  "memory_candidates": []
}
```

---

## DailyPlan

Used by `PlanAgent` to generate the next-day study plan.

```json
{
  "plan_date": "",
  "tasks": [
    {
      "subject": "",
      "task": "",
      "reason": "",
      "estimated_minutes": 0,
      "related_problem_id": "",
      "status": "pending"
    }
  ]
}
```
